# Worked examples — country prompts library

Pre-built examples for the most likely IPTV expansion markets. Use as starting templates; substitute the brand name and any user-overridden inputs before delivering.

Defaults assumed for all: brand-letter from first letter of brand name, gold `#E8B547` accent, navy `#0B1620` background, play-arrow / pennant motion cue.
